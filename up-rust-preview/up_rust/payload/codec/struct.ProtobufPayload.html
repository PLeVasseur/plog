Struct ProtobufPayload

Source 
pub struct ProtobufPayload;
Expand description

Protocol Buffers application payload codec.

ProtobufPayload serializes and deserializes only application payload bytes. It does not wrap a complete uProtocol frame and does not serialize frame metadata.
