Expand description

Initializes a typed value directly in uninitialized transmit storage.

§Safety

Implementors must guarantee that LoanUninitPayload::loan_uninit_payload validates the destination range and only returns a slot when the range is one uniquely borrowed allocation, has the exact layout returned by LoanUninitPayload::loan_uninit_layout, and is valid for writes of one MaybeUninit<T>.
