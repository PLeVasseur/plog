Struct StableContainerPayload

Source 
pub struct StableContainerPayload<T>(/* private fields */);
Expand description

Transport-independent stable-container payload identity.

USR-04A uses this type for metadata and owned-byte preservation. Typed zero-copy borrows are intentionally deferred to the stable borrow proof API.
