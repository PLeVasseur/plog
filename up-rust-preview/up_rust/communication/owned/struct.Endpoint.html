    T: UOwnedTransport + ?Sized,
    P: LocalUriProvider + ?Sized,
{ /* private fields */ }
Expand description

Front door for owned native-frame communication-layer clients.

Role: the up-L2 role set over the owned-frame family — same role semantics as the UTransport-family layer, carrying validated frames instead of messages. See the guide.
