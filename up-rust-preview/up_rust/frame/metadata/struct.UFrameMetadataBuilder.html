Struct UFrameMetadataBuilder

Source 
pub struct UFrameMetadataBuilder { /* private fields */ }
Expand description

Builder for native frame metadata.

Obtained from UFrameMetadata::publish, UFrameMetadata::notification, UFrameMetadata::request, or UFrameMetadata::response. Native selected-wire code should construct metadata through this builder instead of going through UMessageBuilder and projecting.
